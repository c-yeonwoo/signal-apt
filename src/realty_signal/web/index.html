계산식 설명 (입찰가 산정 모델)
• 경매 총매입 = 입찰가 + 등기비(입찰가×취득세율+법무비) + 명도비(전용평×15만) + 미납관리비 + 수리비 + 보유이자(대출금×금리×개월/12)
• 일반매매 총매입 = 시세 + 취득세 + 중개수수료 + 법무비
• 시세차익률 = (일반매매총매입 − 경매총매입) ÷ 경매총매입  (낙찰받아 시세로 되팔 때 차익률)
• 권장 입찰가 = 낙찰가율을 1%씩 올리며 만든 표에서 목표 시세차익률을 만족하는 최대 입찰가
• 실투자금 = 경매총매입 − 대출금(입찰가×대출비율)  |  임대수익률 = (월세×12 − 연이자) ÷ (실투자금 − 임대보증금)  |  단기매도수익률 = (매도가 − 경매총매입 − 매도중개보수) ÷ 실투자금
※ 시세 미입력 시 감정가로 대체. 위 파라미터(목표차익률·대출·보유개월)를 바꾸면 즉시 재계산됩니다.

관심 매물 — 우선순위순

단지 클릭 시 상세(비용·수익·급지) 드릴다운 + 지도 핀 포커스. 상세의 '대출계산기'로 민감도 표.

저평가 지역 추천 — 지역 급지 대비 입지·가격

지역을 평단가 분위로 급지(상~하급지)로 나누고, 각 급지 안에서 입지(교통·학군·환경) 대비 저평가된 곳을 추천합니다. 입지점수 = 업무지구(강남·광화문·여의도) 대중교통 + 학원밀도 + 주거환경. 같은 급지인데 입지 대비 싸면 저평가. ※ 시군구 단위 숏리스트 — 단지별 신축/구축 차이는 반영 안 됨.

수도권 급지 지도 — 색이 진할수록(금색) 상급지. 구를 클릭하면 아래 해설로 이동합니다.

시세(중위호가) 대비 싸게 나온 매물을 지역별로 모읍니다. 급매갭 = (호가−중위시세)/중위시세, 음수가 클수록 저가.

시그널 APT란?

데이터로 찾는 아파트 매매 타이밍. 전문가 상담 없이도, 시장 지표와 흐름을 스스로 보고 매수 의사결정을 내릴 수 있게 돕는 서비스입니다.

KB 주간지수·국토부 실거래·청약홈·경매·정비사업 등 공개 데이터를 우리 방식으로 가공해 매수/매도 시그널을 냅니다. 감이 아니라 과거 구간으로 검증된 적중률로 말합니다.

🌀
경기 국면 가이드 회복·상승·후퇴·침체 4국면과 자산격차. 사이클 읽는 법.
📖
용어사전 전세가율·DSR·비례율 등 매매·투자에 꼭 필요한 개념.
